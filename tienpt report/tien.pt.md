# Personal report

## Data Generation

Trong project này, sử dụng Faker.js để generate data giả lập cho các bảng trong hệ thống.

Để bắt đầu, chạy các lệnh sau: 
```sh
npm install
# node <filename>, for example:
node recruitment.js
```
